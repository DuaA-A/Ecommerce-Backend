For each microservice:
1. Move service to: C:\Ecommerce\service_name\
2. Open that folder in VS Code
3. python -m venv venv
4. .\venv\Scripts\activate
5. pip install flask requests mysql-connector-python python-dotenv
6. pip freeze > requirements.txt
7. Create .env (DB settings)
8. Create db.py
9. Create app.py
10. python app.py
-------------------------------------

DB_PORT=3303
